# VelocityIQ — The Used Car Profit Engine (Demo)

A self-contained, offline product demo. **No install, no internet, no setup.**

---

## ▶ How to open (one click)

| Your computer | Double-click this file |
|---------------|------------------------|
| **Windows**   | `Open VelocityIQ Demo.bat` |
| **macOS**     | `Open VelocityIQ Demo.command` |
| **Linux**     | `open-velocityiq-demo.sh` |

That's it — the demo opens in your default web browser.

> **No launcher?** You can always just double-click **`velocityiq.html`** directly.

> **macOS / Linux first run:** if double-click is blocked, open Terminal in this
> folder and run `chmod +x "Open VelocityIQ Demo.command"` (or the `.sh`), then try again.

---

## 🧭 What to try in the demo

- **Search anything** — press **Ctrl + K** (⌘ + K on Mac) to jump to any page, vehicle, or customer.
- **Recon Tracker → All Units** — expand a vehicle and click **“Advance to …”**. Push it to
  **Ready** and watch the 🔒 red lock flip to 🔓 green — it instantly shows up in **Customer Match**.
- **Recon Tracker → Scan New VIN** — hit **Simulate Scan**, change the **Work Type** or **Vendor**,
  and the **AI predicted ready date** updates live.
- **Dashboard** — the floorplan cost ticker runs every second; the aging buckets show profit at risk.
- **Settings → Floorplan Rate** — change the numbers; the ROI recalculates instantly.
- Your changes **persist** — refresh the page and everything is still there. Reset any time from
  **Settings → Subscription & Billing → “Reset demo data.”**
